# Дикий Запад: Скрытый Фронтир

## Быстрый старт

В игре две наиболее важные команды, которых достаточно, чтобы начать играть.

Первая команда инициирует дуэль:
```
Я самый быстрый стрелок на диком западе!
```

Вторая отвечает на нее. 
```
Нет, я.
```

Отвечать нужно игроку, а не боту.

По умолчанию у игрока есть:
- стартовое оружие `«ржавый револьвер»`, который полностью заряжен (3/3)
- 3 запасных патрона

## Команды
### Правила написания команд
#### Одна команда
Если в комментарии было написано 2 и более команд, отработает только одна.

#### Маркер
Большая часть команд нуждается в маркере перед сообщением (чтобы на каких-то обычных фразах бот не срабатывал лишний раз)

Основные маркеры:

- wwfo
- дикий запад

Почти все команды имеют вариации, но есть и те, что имеют только один вариант.

При общении с ботом в личных сообщениях (только в телеграм) маркер писать необязательно.

#### Новая строка
Команды, которым не нужен маркер, нужно писать с новой строки.

#### Исключения
Все исключения из вышестоящих правил отражены или будут отражены в справке по командам

# Дуэль
Дуэль – это основная активность в игре. Принимая участие в дуэлях, игрок может получить награду в виде внутриигровых денег (UFD – United Frontier dollar).

## Общие правила дуэлей
Начинать или отвечать на дуэль можно только будучи живым
Начать новую дуэль можно раз в час
Одна дуэль может быть сыграна лишь один раз
Ответить можно на любое количество чужих дуэлей
Открытая дуэль перестает быть активной через 8 часов, если на нее никто не ответил
Если начавший дуэль умер вне дуэли, на эту дуэль нельзя ответить
Конец дуэли
Когда дуэль закончена, на нее больше нельзя ответить. Дуэль считается законченной, если:

Дуэль произошла между двумя разными игроками
Дуэль начал и принял один и тот же игрок
Ничья
Ничья – это когда оба игрока попали друг в друга. Если при этом, кто-то умер, это не имеет значения. Награду получают оба игрока: базовая награда / 2

## Ранение
Игрок, в которого выстрелили, получает случайное ранение.
- вид ранения влияет только на количество потерянного HP
- на награду победителя в дуэли это не влияет

## Выстрел в дуэли
Каждый выстрел в дуэли тратит патрон в револьвере. Проигравший в дуэли не стреляет — соответственно не тратит патрон.

## Награда за победу в дуэли
При победе в дуэли игрок получает награду.
- минимальная награда - 10$
- если у победителя до дуэли было меньше HP, чем у соперника, он получит больше денег. Максимально: $100
- если соперник проиграл по причине отсутствия патронов, награда победителя уменьшится, минимальная: $3
- при подсчете награды не имеет значения количество нанесенного урона

## Забронированное предложение на дуэль
Игрок может забронировать место соперника в дуэли, используя особый формат команды.

- можно вызвать только игрока с рангом Ковбой и выше
- остальные ограничения соответствуют обычной дуэли

### Команда

Надо отвечать на комментарий игрока, которого планируется вызвать, используя команду:

```
Тебе пришла пора понять, что я самый быстрый стрелок на Диком Западе!
```

#### Примечания

- отвечать на такую дуэль нужно аналогично обычной: правила не меняются
- игрок может вызывать других, даже если его собственный ранг ниже "Ковбоя"
- другие игроки смогут сдаться на такую дуэль, но это не завершит дуэль, согласно общим правилам дуэлей
- вместо "я самый быстрый стрелок на Диком Западе!" можно использовать любую вариацию стандартного вызова на дуэль

# Здоровье
- стартовое здоровье игрока – 100 HP (health points)
- здоровье ниже или равное 0 означает, что игрок мертв


### Восстановление здоровья

- можно купить аптечку, которая сразу при покупке восстановит ваше здоровье на 70 HP (нельзя купить будучи мертвым).
- время возрождения после смерти – 20 часов, после чего у игрока будет 100 HP

### Смерть
При смерти игрок теряет деньги. Чем дольше игрок живет, тем меньше теряет. 

- $10 – минимальная сумма, которую теряет игрок. По истечении 8 часов с момента последней смерти игрок будет терять именно эту сумму.
- $15 – максимальная сумма, которую может потерять игрок

### Влияние здоровья на награду
Здоровье игрока и его соперника влияют на вероятность выигрыша и на награду. Чем ниже здоровье одного игрока по сравнению с другим, тем меньше его шанс выиграть, но выше награда.

### На что еще влияет здоровье

Чем меньше здоровье, тем:

- меньше денег при поиске грошей в салуне
- меньше выигрыш, если вы победили человека без патронов

# Информация об игроке

Игрок может попросить бота выдать информацию о себе:

- «wwfo мой статус» – здоровье, деньги, активное оружие и другое
- «wwfo мой инвентарь» – информация о предметах в вашем инвентаре
- «wwfo моя статистика» – статистика игрока

### Ранг
Ранг зависит от количества дуэлей, в которых поучаствовал игрок. Список текущих рангов

- Житель/Citizen: 0+
- Ковбой/Cowboy: 20+ дуэлей
- Стрелок/Gunslinger: 100+ дуэлей

На данный момент ранг нигде не отображается.


### Пол игрока
Пол влияет на все фразы, где есть изменение по роду (его/её, выстрелил(а)). Не влияет на склонение ников.

Ограничение: пол можно менять раз в 150 часов

#### Команды:

```
wwfo сменить пол на женский
```

```
wwfo сменить пол на мужской
```

# Оружие
В игре имеется несколько различных единиц оружия. Изначально каждый игрок обладает «ржавым револьвером», экипированным и полностью заряженным.

Игрок может купить новое оружие, после чего его необходимо экипировать, например:

wwfo купить Уолкер
wwfo экипировать Уолкер
Характеристики и цену оружия можно посмотреть в таблице

## Патроны
При каждом выстреле игрок тратит заряженный в экипированное оружие патрон. 

## Перезарядка

Перезарядка происходит автоматически, при перезарядке может произойти случайное осложнение (повреждение руки)
- нужно покупать патроны, после чего тоже произойдет автоперезарядка

- после того как игрок выстреливает последний заряженный патрон, происходит перезарядка
- перезарядка происходит только при наличии патронов в инвентаре
- после перезарядки может произойти случайное «осложнение перезарядки»

Вероятность осложнения перезарядки зависит от характеристики револьвера «удобство перезарядки». Чем выше удобство, тем меньше вероятность ошибки.

## Отсутствие патронов в инвентаре

При отсутствии патронов в инвентаре:
- перезарядка не произойдет, когда закончатся заряженные патроны после выстрела
- покупка патронов вызовет автоперезарядку

## Элитное оружие

Если победивший в дуэли игрок был экипирован элитным револьвером, то перед выстрелом будет добавлена уникальная для такого оружия фраза.

На данный момент единственным элитным оружием является Золотой Питон.


# Магазин

Покупки в игре осуществляются посредством внутриигровой валюты UFD (United Frontier dollar)/$.

Внимание! При покупке аптечки вы сразу ее применяете, в инвентарь она не кладется.

## Общие правила

Команда для покупки в общем случае выглядит так:

wwfo купить ...

Вместо "..." нужно написать название предмета, список которых можно найти в таблице с Магазином.

Для совершения покупки игрок должен быть живым 

## Указание количества при покупке предмета

Некоторые предметы можно покупать в определенном количестве.

Пример:

```
wwfo купить мяч для гольфа x10
```

Вы купите 10 мячей для гольфа, если у вас хватает денег.

Можно указывать количество при покупке следующих предметов: 

- патрон
- мяч для гольфа

Все остальное покупается по одному предмету за раз.

Внимание: модификатор 'x' нужно писать перед числом и слитно. Если написать лишь число, то команда его не учтет и будет распознана как покупка одного предмета. Это служит своеобразным подтверждением со стороны игрока.

Заметим, что следующие команды идентичны:

- wwfo купить один патрон x1
- wwfo купить один патрон
- wwfo купить патрон х1
- wwfo купить патрон

Также идентичными командами являются:

- wwfo купить патроны
- wwfo купить пачку патронов


# Иная активность

## Охота за головами

Игроки могут ставить друг на друга награды за голову. Если игрок с наградой недавно писал комментарий на сайте, на него можно напасть. Если получается при этом его убить, то нападавший забирает награду.

Подробнее про охоту за головами на отдельной странице.

## Обыскать салун
Игрок может получить несколько долларов раз в 6 часов. На размер найденного влияет ваше текущее здоровье

```
wwfo обыскать салун
```

## Отдохнуть

Игрок может восстановить 25 HP один раз в 8 часов. Если у игрока полное здоровье, он отдохнет просто так и потратит возможность. 

```
wwfo отдохнуть
```

## Выстрелить просто так

Игрок может выстрелить просто так

```
wwfo выстрелить вверх
```
```
wwfo выстрелить просто так
```

# Частые ошибки игроков

### Отвечать боту

Иногда игроки пытаются принять дуэль, отвечая «Нет, я» боту, а не игроку. Это не будет работать.

### Редактировать сообщение с командой боту

DTF присылает только события о новых комментариях. Отредактированные сообщения бот уже не получит, поэтому рекомендуется писать команды внимательнее.
